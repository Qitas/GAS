
# STM8

## 描述

STM8系列是ST公司特有架构的8位单片机，同STM32系列有类似的分类分级，可用于简单的边缘节点进行相应的数据采集，在一定应用场景下，可用来替代STM32方案降低BOM成本

---
## 组成

####  connectoSTM8

包含了SI4432的通信工程，可直接用于物联网开发


####  STM8S

STM8通用系列

####  STM8L

STM8低功耗系列

---
## 说明

STM8的开发依赖IAR FOR STM8，开发环境不通用，在我看来会影响开发者学习热情，加上价格差异不够明显，STM32等32位处理器成本下探，不太看好后续的发展。
